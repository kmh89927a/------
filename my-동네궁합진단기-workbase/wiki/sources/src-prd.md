## 핵심 요약

동네궁합진단기의 제품 전략, 타깃 페르소나, 핵심 문제 정의, MoSCoW 기능 우선순위, 북극성 KPI, 롤아웃 계획을 정의하는 최상위 기획 문서. MVP는 3040 맞벌이 부부의 "두 동선 교차 + 배우자 공유 + 데드라인 모드 + 싱글 모드"에 집중한다.

## 주요 내용

### 1. 핵심 가설

> "맞벌이 부부는 두 사람의 직장 동선·학군·생활 편의를 동시에 최적화하는 동네를 효율적으로 찾을 수 있다면, 이사 의사결정 시간이 극적으로 단축된다."

### 2. 타깃 페르소나 (우선순위)

1. **C-01 맞벌이 부부** — 두 직장 + 학군 교차 (김지영, 박상민, 최유리)
2. **C-03 긴급 이사자** — D-2개월 데드라인 (정우진)
3. **C-04 반복 이사자** — 2년 주기 발령 (이수현)
4. **A-01 이직 후 이사 싱글** — 직장 + 여가 거점 (이준혁)

### 3. MoSCoW 우선순위

| 등급 | 기능 |
|---|---|
| **Must** | 두 동선 교차 진단(F1), 배우자 공유 링크(F2), 데드라인 모드(F3) |
| **Should** | 싱글 모드(F4), 간이 저장(F5) |
| **Could** | AI 요약, 학교 배정 구역 레이어 |
| **Won't (MVP)** | 결제, 매물 크롤링, 비교 뷰, 비수도권 |

### 4. 북극성 KPI

- 🌟 진단 완료 수/주: 50 → 200건
- WTP 설문 응답률 ≥ 30%
- 배우자 공유 링크 클릭률 ≥ 40%
- 공유 → 2nd 유저 전환율 ≥ 15%
- NPS ≥ 50

### 5. 롤아웃 계획

| Phase | 대상 | 규모 | 기간 |
|---|---|---|---|
| Alpha | 내부 팀 + 지인 | 10명 | 2026-07 |
| Closed Beta | 맘카페 선발 | 30명 | 2026-08 |
| Open Beta | 수도권 3040 | 300명 | 2026-09~10 |
| GA | 전체 | 1,000명/월 | 2026-11~ |

### 6. Out of Scope

- 결제 시스템 → WTP 설문으로 사전 검증
- 매물 DB 직접 구축 → 네이버 부동산 아웃링크
- 서버 PDF → window.print()
- 비수도권 커버리지
