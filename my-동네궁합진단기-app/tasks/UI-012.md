## 1. 🎯 Summary

- **기능명:** [UI-012] 싱글 모드 진단 화면 UI — 직장+여가 거점 입력 + 야간 치안/편의시설/카페 레이어 토글 + 학군·가족 항목 자동 숨김
- **목적:** 1인 가구(A-01) 사용자가 직장과 여가 거점 2곳을 입력하고, 야간 치안·편의시설·카페 레이어를 토글하여 맞춤 진단을 시작할 수 있는 화면. 학군·가족 관련 필드는 노출 0건 강제. REQ-FUNC-021 준수.
- **범위:**
  - ✅ 싱글 모드 입력 폼 (react-hook-form + Zod, UI-002 패턴 활용), 3개 레이어 토글 (shadcn Switch — 야간 치안/편의시설/카페), 학군·가족 필드 노출 0건 정적 검증, CMD-SINGLE-001 `startSingleModeDiagnosis` 호출, UI-003 MapCanvas 활용, 수도권 검증 (CMD-DIAG-007)
  - ❌ 야간 안전 등급 산출 (QRY-SINGLE-001), 리포트 저장 (CMD-SINGLE-002/UI-013), 결제, NextAuth.js
- **복잡도:** M | **Wave:** 5 (UI 트랙)

  export const singleModeSchema = z.object({
    workplace: z.string().min(1, '직장 주소를 입력해주세요'),
    leisure: z.string().min(1, '여가 거점 주소를 입력해주세요'),

- [ ] **3.3** `components/single/SingleModeForm.tsx` — 입력 폼 + 레이어 토글
  ```typescript
  'use client';
  import { useForm } from 'react-hook-form';
  import { zodResolver } from '@hookform/resolvers/zod';
  import { singleModeSchema, type SingleModeFormData } from '@/lib/validators/single-mode';
  import { Switch } from '@/components/ui/switch';
  import { Label } from '@/components/ui/label';
  import { Input } from '@/components/ui/input';
  import { Button } from '@/components/ui/button';
  import { startSingleModeDiagnosis } from '@/app/actions/single-mode';
  import { useRouter } from 'next/navigation';
  import * as Sentry from '@sentry/nextjs';

  export function SingleModeForm() {
    const router = useRouter();
    const { register, handleSubmit, watch, setValue, formState: { errors, isSubmitting } } = useForm<SingleModeFormData>({
      resolver: zodResolver(singleModeSchema),
      defaultValues: { filters: { maxCommuteTime: 60, nightSafety: true, facilities: true, cafes: true } },
    });

    async function onSubmit(data: SingleModeFormData) {
      try {
        const result = await startSingleModeDiagnosis({
          addressA: data.workplace,
          coordA: { lat: 0, lng: 0 }, // geocoding 결과로 대체
          filters: data.filters,
          candidates: [],
        });
        router.push(`/diagnosis/${result.id}`);
      } catch (e) {
        Sentry.captureException(e);
      }
    }

    return (
      <form onSubmit={handleSubmit(onSubmit)} className="space-y-6 font-pretendard" aria-label="싱글 모드 진단 입력">
        {/* 직장 주소 */}
        <div className="space-y-2">
          <Label htmlFor="workplace">🏢 직장 주소</Label>
          <Input id="workplace" {...register('workplace')} placeholder="예: 강남역" aria-required="true" />
          {errors.workplace && <p className="text-sm text-destructive" role="alert">{errors.workplace.message}</p>}
        </div>
        {/* 여가 거점 */}
        <div className="space-y-2">
          <Label htmlFor="leisure">🎯 여가 거점</Label>
          <Input id="leisure" {...register('leisure')} placeholder="예: 홍대입구역" aria-required="true" />
          {errors.leisure && <p className="text-sm text-destructive" role="alert">{errors.leisure.message}</p>}
        </div>
        {/* 학군·가족 필드 — 절대 노출 금지 (REQ-FUNC-021) */}
        {/* ⚠️ schoolDistrict, familySize, schoolZone, childAge 필드 0건 */}
        {/* 레이어 토글 */}
        <fieldset className="space-y-3" aria-label="지도 레이어 설정">
          <legend className="text-sm font-medium mb-2">지도 레이어</legend>
          <div className="flex items-center justify-between">
            <Label htmlFor="night-safety">🌙 야간 치안</Label>
            <Switch id="night-safety" checked={watch('filters.nightSafety')} onCheckedChange={(v) => setValue('filters.nightSafety', v)} aria-label="야간 치안 레이어 토글" />
          </div>
          <div className="flex items-center justify-between">
            <Label htmlFor="facilities">🏪 편의시설</Label>
            <Switch id="facilities" checked={watch('filters.facilities')} onCheckedChange={(v) => setValue('filters.facilities', v)} aria-label="편의시설 레이어 토글" />
          </div>
          <div className="flex items-center justify-between">
            <Label htmlFor="cafes">☕ 카페</Label>
            <Switch id="cafes" checked={watch('filters.cafes')} onCheckedChange={(v) => setValue('filters.cafes', v)} aria-label="카페 레이어 토글" />
          </div>
        </fieldset>
        <Button type="submit" disabled={isSubmitting} className="w-full" id="btn-single-start">
          {isSubmitting ? '진단 중...' : '진단 시작'}
        </Button>
      </form>
    );
  }
  ```

## 9. 🚧 Open Questions / Risks (보류 사항)

1. **여가 거점 자동완성 데이터:** 카카오 Local API 카테고리 활용 여부. 현재는 텍스트 입력 + Geocoding. 카페/맛집 카테고리 자동완성은 follow-up.
2. **Switch 기본값 ON vs OFF:** REQ-FUNC-021은 "기본 활성화" 명시 → 3개 모두 ON. 사용자가 OFF로 변경 시 해당 레이어 지도에서 제거.
3. **MapCanvas 싱글 모드 통합:** UI-003 MapCanvas가 커플 모드 기반. 싱글 모드에서 단일 주소(직장) + 여가 거점 마커 2개 표시로 분기 필요.
4. **직장·여가 동일 주소:** 직장과 여가 거점이 같은 주소일 때 교집합 로직이 정상 동작하는지 확인 (CMD-SINGLE-001 Open Questions #3).
5. **최대 통근 시간 슬라이더:** UI-005 패턴 활용 시 디바운스 적용 여부. 현재 스키마에 `maxCommuteTime` 포함.
