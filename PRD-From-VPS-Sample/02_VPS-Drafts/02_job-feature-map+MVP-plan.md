MVP (즉시 0~3개월): 두 동선 교차 지도 · 배우자 공유 링크 · 긴급 데드라인+급매 연동 · 싱글 모드 리포트 → 4개 기능이 AOS/DOS 상위를 독점.

v1 (3~6개월): 광역버스 착석 API · 입력값 자동 저장 · 야간 치안 레이어 · 계약 후 가이드 콘텐츠 → 세그먼트별 락인 강화.
