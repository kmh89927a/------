# 페르소나 스펙트럼 — 12명의 사용자 유형

## 정의

서비스의 잠재 사용자를 Core → Adjacent → Extreme → Non-user 4계층 12명으로 분류한 스펙트럼. 각 페르소나는 서비스의 핵심 기능과 대응됨.

## Core User (5명) — MVP 타깃

| # | 이름 | 나이 | 유형 | 대응 기능 | 세그먼트 코드 |
|---|---|---|---|---|---|
| ① | 김지영 | 36 | 워킹맘 (남편 판교 + 본인 여의도) | F1 교차 진단 + F2 공유 | C-01 |
| ② | 박상민 | 41 | 맹모 아빠 (대치 학군 + 광역버스) | F1 + 학원가 레이어 | C-02 |
| ③ | 최유리 | 33 | 예비신부 (강남+잠실+GTX 호재) | F1 + 호재 필터 | C-01 |
| ④ | 정우진 | 38 | 긴급 이사 (D-2개월) | F3 데드라인 모드 | C-03 |
| ⑤ | 이수현 | 39 | 교사 (2년 주기 발령) | F5 저장 + 재사용 | C-04 |

## 설계 원칙

- Core 5명의 Pain Point를 모두 해결하면 MVP 가치 검증 완료
- Adjacent ⑥은 싱글 모드로 MVP에 포함, ⑦⑧은 v2
- Non-user는 마케팅 예산 집행 불필요, 포지셔닝 역참고 자료
