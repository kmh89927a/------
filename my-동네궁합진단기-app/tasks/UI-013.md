## 1. 🎯 Summary

- **기능명:** [UI-013] 야간 안전 등급(A~D) 표시 UI + 리포트 저장(window.print) 버튼
- **목적:** 싱글 모드 후보 동네별 야간 안전 등급(A~D)을 색상+텍스트로 시각화하고, 리포트 저장 버튼으로 window.print() PDF 저장을 제공한다. 색맹 대응(색상+텍스트 동시)과 인쇄 시 버튼 숨김(@media print)을 강제한다.
- **범위:**
  - ✅ `SafetyGradeBadge` 컴포넌트 (A~D 색상 코딩 + 텍스트), `SafetyGradePanel` (후보 동네별 등급 목록), CMD-SINGLE-002 `SaveReportButton` 통합, @media print CSS 정합성 (.no-print), a11y (aria-label, 색맹 대응)
  - ❌ 야간 안전 등급 산출 (QRY-SINGLE-001 범위), window.print 구현 (CMD-SINGLE-002 범위), 결제, NextAuth.js
- **복잡도:** L | **Wave:** 5 (UI 트랙)

- [ ] **3.1** `components/single/SafetyGradeBadge.tsx` — 등급 배지 컴포넌트
  ```typescript
  'use client';
  import { Badge } from '@/components/ui/badge';
  import type { SafetyGrade } from '@/lib/single-mode/safety-grade';

  const GRADE_STYLES: Record<SafetyGrade, { bg: string; text: string; label: string }> = {
    A: { bg: 'bg-green-500 hover:bg-green-600', text: '매우 안전', label: '야간 안전 A등급 — 매우 안전' },
    B: { bg: 'bg-blue-500 hover:bg-blue-600', text: '안전', label: '야간 안전 B등급 — 안전' },
    C: { bg: 'bg-yellow-500 hover:bg-yellow-600', text: '주의', label: '야간 안전 C등급 — 주의' },
    D: { bg: 'bg-red-500 hover:bg-red-600', text: '위험', label: '야간 안전 D등급 — 위험' },
  };

  export function SafetyGradeBadge({ grade }: { grade: SafetyGrade }) {
    const style = GRADE_STYLES[grade];
    return (
      <Badge
        className={`${style.bg} text-white grade-badge`}
        aria-label={style.label}
        role="status"
      >
        야간 안전 {grade}등급 · {style.text}
      </Badge>
    );
  }
  ```

## 4. ✅ Acceptance Criteria (GWT 패턴, BDD)

**AC-1 (정상):** 후보 동네별 A~D 등급 표시
- **Given** 싱글 모드 진단 완료, 후보 동네 3곳
- **When** SafetyGradePanel 렌더링
- **Then** 3곳 각각에 SafetyGradeBadge (A/B/C/D 중 하나) 표시

**AC-2 (도메인 핵심):** 색상 + 텍스트 동시 (색맹 대응)
- **Given** SafetyGradeBadge
- **When** 렌더링
- **Then** 배지에 "야간 안전 A등급 · 매우 안전" 형태로 색상 + 텍스트 동시 표시. 텍스트만으로도 등급 구분 가능

**AC-3 (정상):** print 버튼 클릭 → window.print() 호출
- **Given** SingleReportView 렌더링 완료
- **When** "리포트 저장 (PDF)" 버튼 클릭
- **Then** `window.print()` 1회 호출, 브라우저 인쇄 다이얼로그 표시

**AC-4 (도메인 핵심):** 인쇄 시 print 버튼 숨김 (@media print)
- **Given** @media print CSS 활성
- **When** 인쇄 레이아웃 적용
- **Then** `.no-print` 클래스의 SaveReportButton 컨테이너 `display: none`, 본문 콘텐츠만 인쇄

**AC-5 (도메인 핵심):** a11y — aria-label 등급 의미 설명
- **Given** SafetyGradeBadge `grade='A'`
- **When** 스크린 리더 접근
- **Then** `aria-label="야간 안전 A등급 — 매우 안전"` 읽기

**AC-6 (예외):** 등급 데이터 없음 시 D등급 + "데이터 없음" 표시
- **Given** 비수도권 좌표로 인해 nearby 0건
- **When** SafetyGradeBadge grade='D'
- **Then** "야간 안전 D등급 · 위험" 표시 (QRY-SINGLE-001 반환값 기반)

**AC-7 (경계):** 인쇄 시 등급 배지 흑백 대응
- **Given** @media print CSS
- **When** 흑백 프린터 출력
- **Then** `.grade-badge` → `border: 1px solid #000` 규칙으로 배지 경계 유지
